# Urlo Creativo — Copy Formatting Guide

| Write this | Renders as |
|---|---|
| `**word**` | **Bold** |
| `*word*` | *Italic* |
| `***word***` | ***Bold + Italic*** |
| `\n` | Line break |

Use markers exactly where shown in the copy template. Don't add or remove them — only change the words inside.
