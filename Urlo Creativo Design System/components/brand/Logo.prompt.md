The Urlo Creativo logo — use the **wordmark/lockup** in page headers and the **mark** alone as a compact nav glyph. Image-based; set `basePath` to point at `assets/`.

```jsx
<Logo variant="lockup" height={52} basePath="../../" />
<Logo variant="mark" height={32} basePath="../../" />
```

Variants: `lockup` (mark + type), `wordmark`, `mark`. Never recolor or add effects — the studio uses it flat on paper white.
