The house pill button. **Outline** is the default (2px ink ring, fills with ink on hover); **solid** for primary moments; **ghost** for low-emphasis (turns yellow on hover).

```jsx
<Button>Contact</Button>
<Button variant="outline" size="sm">Discover our services</Button>
<Button variant="solid" size="lg">Book a consultation</Button>
```

Sizes: `sm` / `md` / `lg`. Render as a link with `as="a" href="…"`. Keep labels sentence-case and short.
